Unity plugin - A Pathfinding Project Pro

One-minute read
Unity plugin
plugin

相关插件 A* pathfinding project pro

实现在Unity上的AStarPathfinding寻路的插件，提供了高级的寻路AI系统;

  • 新建空物体命名为"A*"，并添加pathfinder组件脚本;

  • 添加Graph，选择第一种类型Grid Graph; )

    • Shap – 寻路网格类型

    • width 和 depth – 表示寻路网格面积区域;

    • Node size – 单个寻路网格单位尺寸

    • Height testing

      • ray length – 从上自下的发射检测射线，以此来确定此网格单位表面是否可寻址区域; 通过移动来对比出经过射线检测后寻路网格区域:

    • Collision Testing

      • Obstacle Layer Mask – 添加障碍物层，属于此层的物体将不会在其表面生成寻路网格:

  • 添加寻路AI，可以从示例场景中拷贝bot对象;
