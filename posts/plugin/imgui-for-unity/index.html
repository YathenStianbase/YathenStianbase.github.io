Imgui for Unity [Unity版本Imgui]

One-minute read
Unity plugin
imgui • plugin

  • 安装方式

    • 使用git url方式安装
      • 原版: https://github.com/realgamessoftware/dear-imgui-unity.git
      • 镜像: https://hub.nuaa.cf/realgamessoftware/dear-imgui-unity.git
      • 硬盘: 拷贝其他项目下的缓存目录，package安装方式选择 “Add pacakge from disk”,选中缓存目录中的package.json文件

  • 使用方式

    using ImGuiNET; // 

void OnEnable()
{
	ImGuiUn.Layout += OnLayout;
}

void OnDisable()
{
	ImGuiUn.Layout -= OnLayout;
}

void OnLayout()
{
	ImGui.ShowDemoWindow(); // 展示Demo窗口

  ImGui.Text($"显示"); // 测试文本
  if (ImGui.Button("Save")) // 测试按钮
  {
  	Debug.Log("Save");
  }
}

    • CreateWindow(ImGuiWindowFlags.NavFlattened); 此方法的使用会导致unity2019.4奔溃;
