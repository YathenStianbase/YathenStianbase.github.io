Android ADB Tool
ADB
  • 命令:
    • 查看帮助： 
      adb --help
    • 查看连接设备： 
      adb devices
    • 开启无线网络连接： 
      adb tcpip port （默认5555）
    • 使用无线网络连接： 
      adb connect ip_address
    • 连接shell： 
      adb -s 指定设备 shell
    • 倒出配置： 
      dumpsys
      使用dumpsys 命令查找app启动入口(注意 Activity Resolver Table: 下的 android.intent.action.MAIN: 之后一般包含的就是启动activity);
    • 启动app： 
      am start {包(package)名}/{包名}.{活动(activity)名称}
