AndroidSDK sdkmanager [sdkmanager 命令工具]

One-minute read
Android SDK Tool
sdkmanager

  • 在cmd窗口中设置临时javahome路径：

    • set JAVA_HOME=WindowsPath

  • sdkmanager工具路径 .\android-sdk-windows\tools\bin 帮助命令：

    • sdkmanager –help // 查看命令帮助

  • 下载地址(将https改为http,将dl-ssl改为dl) http://dl-ssl.google.com/android/repository/tools_r25.2.5-windows.zip

  • 新建"SDK_ROOT"目录, cd SDK_ROOT\tools\bin

    • sdkmanager –list 列出所有版本

    • sdkmanager “platform-tools” 生成包含相关工具的platform-tools目录

    • sdkmanager “platforms;android-25” 生成包含相关工具的platforms目录

    • sdkmanager “build-tools;25.0.0” 生成包含相关工具的build-tools目录

  • 如果报一下错误， 确保jdk版本对应android sdk版本：
